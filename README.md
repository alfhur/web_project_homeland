# Tripleten web\_project\_homeland





📌 Descripción del proyecto



Este proyecto consiste en el desarrollo de una página web responsiva que muestra las ciudades natales de algunos empleados de TripleTen.





🎯 Objetivo



Aplicar los conocimientos aprendidos en el Sprint 5:



* Desarrollar una página web con diseño responsivo.
* Aplicar media queries.
* Utilizar la metodología BEM con una estructura plana de archivos.
* Optimizar imágenes para mejorar el rendimiento.
* Clonar un repositorio remoto GitHub
* Usar tipografía almacenada localmente
* Optimizar imágenes





✅ Resultado esperado



Una página web completamente responsiva que:



* Respete el diseño proporcionado en Figma.
* Funcione correctamente en dispositivos móviles, tablets y escritorio.
* Mantenga consistencia visual y estructural.
* No presente errores de maquetación entre breakpoints.





📚 Tecnologías utilizadas



* HTML5
* CSS3
* Metodología BEM
* Diseño responsivo
* Figma como herramienta de diseño
